## Akka HTTP Project with REST API
######  REST API, HTTP, distributed programming, 2021
2021.08.01 ~ 2021.08.31
### Dependecnies
* scala 2.11.8
* akka 2.5.26

### Introduction
* akka 분산 및 병렬 프레임워크를 이용하여 액터 기반의 HTTP 웹 서버를 제작한다.
* HTTP 웹서버는 REST API 를 기반으로 한다. 
* 낮은 latency를 보장한다.

### Module
1. my-akka-actor
    * actor 동작 과정 학습용 모듈
2. my-akka-http
    * http 서버 구현용 모
